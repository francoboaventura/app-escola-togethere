# App Escola Togethere - v1

Aplicativo web para gestão escolar com controle de presença, temas de casa, testes, material, planos de aula e tarefas internas.

## 📋 Funcionalidades v1

- ✅ Controle de presença de alunos
- ✅ Registro de temas de casa
- ✅ Resultados de testes
- ✅ Registro de material não trazido
- ✅ Atas de reuniões
- ✅ Planos de aula com checklist
- ✅ Resumo automático de aula para pais
- ✅ Alertas para secretaria (faltas e material)
- ✅ Tarefas internas com matriz de Eisenhower

## 🚀 Como usar

1. Abra `index.html` no navegador
2. Use a sidebar para navegar entre funcionalidades
3. Registre os dados da turma/aula
